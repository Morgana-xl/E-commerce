# E-commerce
Projeto Lógico de Banco de Dados de E-commerce
Desenvolvido como atividade do Bootcamp da DIO
Operações a se destacar: 
Cadastro de clientes e fornecedores;
registro de pedidos e pagamentos;
controle de estoque e entregas.

* Estrutura do Banco de Dados:
O banco de dados foi modelado para conter as seguintes informações:

- **Client2** → Representa os clientes, podendo ser pessoa física (PF) ou jurídica (PJ)
- **Product** → Catálogo de produtos disponíveis para venda
- **Orders** → Pedidos realizados pelos clientes
- **Payment** → Formas e registros de pagamento
- **Delivery** → Controle das entregas e rastreios
- **Supplier** → Fornecedores de produtos
- **Stock** → Estoques e suas localizações
- **Seller** → Vendedores e lojas parceiras
- **Tabelas associativas** → Ligam entidades principais (ex: ProductSupplier, ProductOrder etc.)
